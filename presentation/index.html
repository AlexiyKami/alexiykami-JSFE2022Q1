Typescript

Is a superset of JavaScript

with static typing

object-oriented

Typescript

open-source language

created by Microsoft in 2012

Installation

npm install typescript

Using

  • 1. Create a file called helloworld.ts
  • 2. Write the following code in it: 
    
								const a: string = "hello world";
								console.log(a);
  • 3. Run the tsc command
  • 4. You got a helloworld.js file that you can run

Classes

Declaration: 

							class Point {
								x: number;
								y: number;
								constructor(x: number, y: string) {
									this.x = x;
									this.y = y;
								}
								getX() {
									return this.x;
								}
							}
Create an instance: 

							const pt = new Point();

Static members


							class Circle {
								static pi: number = 3.14;
							}
							Circle.pi; // returns 3.14

Readonly members


							class Foo {
								readonly bar: number = 123;
							}

Interfaces

Declaration: 

							interface User {
								id: number
								name: string;
								age?: number;
							}
